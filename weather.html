A collaborative group project where I led both design and development. I created the final high-fidelity Figma prototype based on team research, then built the functional web platform, coding most of the JavaScript logic and all of the CSS.

The Brief

A responsive weather platform built for Cork, serving different user profiles including farmers, marine users, and outdoor planners. It pulls live forecast data from a real weather API, delivering current conditions, hourly and 7-day forecasts, and a radar map. Users can personalise their dashboard with specialist widgets covering marine conditions, farming indicators, pollen, air quality, and UV, with a chatbot to answer weather queries and help manage alert preferences.

We worked directly with the client throughout, incorporating their feedback at each stage of the process.

My Role

I created the entire Figma prototype from scratch based on the team's research findings, establishing the visual direction and interaction patterns. I then built the functional platform, writing most of the JavaScript logic and all of the CSS, using HTML skeleton components provided by teammates.
